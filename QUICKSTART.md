## Troubleshooting

**Don't see the "Website Auditor" tab?**
- Make sure you selected the `dist` folder (not the parent folder)
- Refresh the extension in `chrome://extensions/`
- Close DevTools and reopen it (F12)

**Scores showing as 0?**
- Reload the webpage
- Run the audit again

**Extension shows error?**
- Make sure all files are in the `dist` folder
- Check that `manifest.json` exists in `dist/`
